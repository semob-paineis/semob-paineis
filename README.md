## 🔧 Como Fazer Deploy no Vercel

### **Passo 1: Conectar GitHub ao Vercel**
1. Acesse https://vercel.com
2. Clique em "Sign up" → "Continue with GitHub"
3. Autorize a conexão

### **Passo 2: Importar Projeto**
1. Clique em "Add New" → "Project"
2. Selecione o repositório `semob-dashboards`
3. Clique em "Import"

### **Passo 3: Deploy**
1. Vercel detecta automaticamente
2. Clique em "Deploy"
3. Aguarde ~1 minuto

### **Pronto!**
Seu painel estará em: `https://semob-dashboards.vercel.app/`
